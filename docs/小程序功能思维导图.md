# 微信小程序功能规划思维导图

## 📱 核心功能模块

### 1. 首页 - 数据概览
- **实时数据显示**
  - 当前皮肤水分含量
  - 当前环境温湿度
  - PM2.5和CO2浓度
  - 设备在线状态
- **快捷入口**
  - 历史数据查询
  - 设备管理
  - 告警中心
  - 个人中心

### 2. 实时监测
- **皮肤传感器**
  - 水分含量趋势图（24小时）
  - 油脂度变化曲线
  - 温度波动图
  - 数据质量评分
- **环境传感器**
  - 温湿度实时监控
  - PM2.5空气质量指数
  - CO2浓度监测
  - 地理位置显示
- **刷新机制**
  - 自动刷新（每30秒）
  - 手动下拉刷新
  - WebSocket实时推送（可选）

### 3. 历史数据
- **时间筛选**
  - 今日/昨日
  - 本周/上周
  - 本月/上月
  - 自定义日期范围
- **数据展示**
  - 折线图趋势分析
  - 柱状图对比
  - 数据表格详情
  - 导出Excel/PDF
- **统计指标**
  - 平均值/最大值/最小值
  - 标准差
  - 异常数据标记

### 4. 设备管理
- **设备列表**
  - 设备基本信息
  - 在线/离线状态
  - 电池电量显示
  - 信号强度指示
- **设备详情**
  - 固件版本信息
  - 安装位置地图
  - 最后心跳时间
  - 运行时长统计
- **设备操作**
  - 添加新设备（扫码/手动输入）
  - 解绑设备
  - 设备重命名
  - 分享设备给他人

### 5. 告警中心
- **告警类型**
  - 水分过低告警
  - 温度异常告警
  - PM2.5超标告警
  - 设备离线告警
  - 低电量告警
- **告警设置**
  - 阈值自定义
  - 告警级别设置
  - 通知方式选择（微信消息/短信）
  - 免打扰时段设置
- **告警历史**
  - 未读/已读标记
  - 告警处理记录
  - 告警统计分析

### 6. 数据分析
- **日报/周报/月报**
  - 自动生成统计报告
  - 关键指标汇总
  - 趋势分析图表
  - 健康建议推荐
- **对比分析**
  - 多设备数据对比
  - 不同时间段对比
  - 同类用户平均对比
- **智能建议**
  - 基于数据的护肤建议
  - 环境改善建议
  - 设备维护提醒

### 7. 个人中心
- **用户信息**
  - 头像/昵称修改
  - 手机号绑定
  - 实名认证
- **我的设备**
  - 已绑定设备列表
  - 设备权限管理
- **设置**
  - 主题切换（深色/浅色）
  - 数据同步设置
  - 隐私政策
  - 关于我们
- **帮助与反馈**
  - 常见问题FAQ
  - 在线客服
  - 意见反馈

## 🔧 技术架构

### 前端技术栈
- **框架**: 微信小程序原生开发 / Taro跨端框架
- **UI组件库**: Vant Weapp / MinUI
- **图表库**: ECharts for Weixin / F2
- **状态管理**: MobX-miniprogram

### 后端接口
- **API网关**: Flask RESTful API
- **认证方式**: JWT Token + 微信OpenID
- **数据源**: 
  - MySQL（统计数据、用户信息）
  - MongoDB（原始传感器数据）
- **实时通信**: WebSocket / 小程序订阅消息

### 数据库设计扩展
```sql
-- 小程序相关表（已在MySQL中创建）
users                    -- 用户表
user_device_bindings     -- 用户设备绑定表
alerts                   -- 告警记录表
daily_statistics         -- 每日统计表
```

## 🎨 UI/UX设计要点

### 设计风格
- **主色调**: 科技蓝 (#1890FF)
- **辅助色**: 成功绿(#52C41A)、警告橙(#FAAD14)、错误红(#FF4D4F)
- **字体**: 苹方/思源黑体
- **图标**: 线性图标为主，填充图标为辅

### 交互设计
- **加载状态**: 骨架屏 + Loading动画
- **空状态**: 友好提示 + 引导操作
- **错误处理**: Toast提示 + 重试按钮
- **手势操作**: 左滑删除、下拉刷新、上拉加载更多

### 响应式设计
- 适配不同屏幕尺寸
- 横屏/竖屏自适应
- 深色模式支持

## 📊 数据流向

```
传感器设备 → MQTT消息队列 → Flask接收 → MongoDB存储
                                    ↓
                              定时任务(APScheduler)
                                    ↓
                              统计计算 → MySQL存储
                                    ↓
                              Flask API接口
                                    ↓
                              微信小程序展示
```
