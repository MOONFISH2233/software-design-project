## 🚀 5 分钟快速部署

### 方式一：使用上传脚本（推荐新手）

1. **在本地 PowerShell 中运行上传脚本**
```powershell
cd "d:\学习\软件设计\data-server"
.\upload-to-server.ps1
```

2. **SSH 登录服务器**
```bash
ssh root@47.103.108.47
```

3. **安装依赖并启动服务**
```bash
cd /root/data-server
pip3 install -r requirements.txt
nohup python3 app.py > server.log 2>&1 &
```

4. **测试服务**
```bash
curl http://localhost:5000/api/health
```

## 📝 配置防火墙

### 阿里云安全组配置

1. 登录阿里云控制台
2. 进入 **云服务器 ECS** → 你的实例
3. 点击 **安全组** → **配置规则**
4. 添加入方向规则：
   - 端口范围：`5000/5000`
   - 授权对象：`0.0.0.0/0`
   - 协议：`TCP`

### 服务器防火墙配置

```bash
# SSH 登录服务器后执行
firewall-cmd --zone=public --add-port=5000/tcp --permanent
firewall-cmd --reload
firewall-cmd --list-ports
```

##  使用示例

### Python 客户端上传数据
```python
import requests

# 上传传感器数据
data = {
    "temperature": 25.5,
    "humidity": 60,
    "location": "实验室",
    "device_id": "sensor_001"
}

response = requests.post(
    'http://47.103.108.47:5000/api/receive',
    json=data
)

print(response.json())
```

### 本地文件上传
```python
from local_to_cloud import DataUploader

# 初始化上传器
uploader = DataUploader('http://47.103.108.47:5000')

# 检查服务器状态
print(uploader.health_check())

# 上传数据
sample_data = {
    'temperature': 25.5,
    'humidity': 60,
    'location': '实验室'
}
uploader.upload_data(sample_data, "环境传感器数据")

# 查看上传历史
for record in uploader.get_upload_history():
    print(record)
```

### 云端同步
```python
from cloud_to_cloud import CloudToCloudTransfer

# 从服务器 A 同步到服务器 B
transfer = CloudToCloudTransfer(
    source_server='http://47.103.108.47:5000',
    target_server='http://47.103.108.47:5000'
)

# 执行同步
result = transfer.sync('/api/logs', "日志数据同步")
print(result)
```

## 🔐 安全建议

1. **修改默认端口**（可选）
   - 编辑 `app.py`，将 `port=5000` 改为其他端口

2. **配置 HTTPS**（生产环境推荐）
   - 使用 Nginx 反向代理
   - 申请 SSL 证书

3. **添加认证机制**
   - 在 API 中添加 token 验证

4. **定期备份数据**
   ```bash
   tar -czf backup_$(date +%Y%m%d).tar.gz /root/data-server/data/
   ```
