## 📊 API 接口文档

### 接口总览

| 接口 | 方法 | 功能 | 传输频率 |
|------|------|------|----------|
| `/api/receive` | POST | 接收并保存数据 | 实时/按需（≥1 秒） |
| `/api/health` | GET | 健康检查 | 定期（30 秒） |
| `/api/logs` | GET | 查看日志 | 按需（≥5 秒） |

### 详细参数

#### 1. POST /api/receive - 数据接收

**请求参数：**
| 参数名称 | 类型 | 必填 | 说明 | 示例值 |
|----------|------|------|------|--------|
| temperature | float | 否 | 温度值 | 25.5 |
| humidity | int | 否 | 湿度值 | 60 |
| location | string | 否 | 位置 | "实验室" |
| device_id | string | 否 | 设备 ID | "sensor_001" |
| data | object | 否 | 任意数据对象 | {...} |
| description | string | 否 | 数据描述 | "环境数据" |
| source | string | 否 | 数据来源 | "local" |
| upload_time | string | 否 | ISO 时间 | "2026-03-22T15:30:00" |

**请求示例：**
```bash
curl -X POST http://47.103.108.47:5000/api/receive \
  -H "Content-Type: application/json" \
  -d '{
    "temperature": 25.5,
    "humidity": 60,
    "location": "实验室",
    "device_id": "sensor_001"
  }'
```

**响应示例：**
```json
{
  "status": "success",
  "message": "数据接收成功",
  "filename": "data_20260322_153000_123456.json",
  "timestamp": "2026-03-22T15:30:00"
}
```

#### 2. GET /api/health - 健康检查

**请求参数：** 无

**响应示例：**
```json
{
  "status": "healthy",
  "service": "Flask Data Server",
  "timestamp": "2026-03-22T15:30:00"
}
```

#### 3. GET /api/logs - 日志查询

**请求参数：** 无

**响应示例：**
```json
{
  "logs": "2026-03-22 15:30:00 - INFO - 成功接收并保存数据...\n...",
  "file": "server_20260322.log"
}
```

## 💻 使用示例

### 示例 1：Python 客户端上传数据

```python
import requests

# 上传数据
data = {
    "temperature": 25.5,
    "humidity": 60,
    "location": "实验室",
    "device_id": "sensor_001"
}

response = requests.post(
    'http://47.103.108.47:5000/api/receive',
    json=data
)

print(response.json())
```

### 示例 2：本地文件上传到云端

```python
from local_to_cloud import DataUploader

# 初始化
uploader = DataUploader('http://47.103.108.47:5000')

# 检查服务器
print(uploader.health_check())

# 上传数据
sample_data = {
    'temperature': 25.5,
    'humidity': 60,
    'location': '实验室'
}
result = uploader.upload_data(sample_data, "环境传感器数据")
print(result)

# 上传文件
# uploader.upload_file('local_data.json', "本地数据文件")

# 查看历史
for record in uploader.get_upload_history():
    print(record)
```

### 示例 3：云端到云端同步

```python
from cloud_to_cloud import CloudToCloudTransfer

# 初始化
transfer = CloudToCloudTransfer(
    source_server='http://47.103.108.47:5000',
    target_server='http://47.103.108.47:5000'  # 可改为另一台服务器
)

# 执行同步
result = transfer.sync('/api/logs', "日志数据同步")
print("同步结果:", result)

# 查看历史
for record in transfer.get_transfer_history():
    print(record)
```

### 示例 4：cURL 命令

```bash
# 健康检查
curl http://47.103.108.47:5000/api/health

# 上传数据
curl -X POST http://47.103.108.47:5000/api/receive \
  -H "Content-Type: application/json" \
  -d '{"temperature": 25.5, "humidity": 60}'

# 查看日志
curl http://47.103.108.47:5000/api/logs

systemctl enable jenkins
```

**2. 访问 Jenkins**

- 初始密码：`cat /var/lib/jenkins/secrets/initialAdminPassword`

**3. 安装插件**
- Git Plugin
- SSH Agent Plugin
- Pipeline Plugin

**4. 配置凭证**
- Manage Jenkins → Manage Credentials
- 添加 SSH Key（服务器登录）
- 添加 Git 凭证

**5. 创建 Pipeline**
- 新建任务 → Pipeline
- Pipeline script from SCM
- 选择 `Jenkinsfile`

**6. 一键部署**
- 点击 **Build Now**

## ❓ 常见问题

### Q1: 服务无法访问？
```bash
# 1. 检查服务是否运行
ps aux | grep app.py

# 2. 检查端口
netstat -tulpn | grep 5000

# 3. 检查防火墙
firewall-cmd --list-ports

# 4. 检查安全组（阿里云控制台）
```

### Q2: 依赖安装失败？
```bash
# 升级 pip
pip3 install --upgrade pip

# 重新安装
pip3 install -r requirements.txt
```

### Q3: 如何修改端口？
编辑 `app.py`，找到：
```python
app.run(host='0.0.0.0', port=5000, debug=True)
```
修改 `port=5000` 为其他端口，如 `port=8080`
