# Flask 数据服务器 - API 接口文档（与实际代码一致）

## 接口列表

### 1. 数据接收接口

| 属性 | 说明 |
|------|------|
| **接口地址** | `/api/receive` |
| **请求方式** | POST |
| **数据格式** | JSON |
| **传输频率** | 实时/按需（建议间隔≥1 秒） |

**请求参数（皮肤传感器示例）**

| 参数名称 | 类型 | 必填 | 说明 | 示例 |
|----------|------|------|------|------|
| sensor_type | string | 是 | 传感器类型，固定为 `"skin"` | `"skin"` |
| moisture | int | 是 | 水分度（0-100） | `65` |
| oiliness | int | 是 | 油亮度（0-100） | `32` |
| timestamp | string | 是 | 数据采集时间，格式 `YYYY-MM-DD HH:MM:SS` | `"2026-03-27 10:11:44"` |

**请求参数（环境传感器示例）**

| 参数名称 | 类型 | 必填 | 说明 | 示例 |
|----------|------|------|------|------|
| sensor_type | string | 是 | 传感器类型，固定为 `"environment"` | `"environment"` |
| humidity | int | 是 | 环境湿度（0-100） | `71` |
| light_lux | int | 是 | 光照度（Lux） | `771` |
| timestamp | string | 是 | 数据采集时间，格式 `YYYY-MM-DD HH:MM:SS` | `"2026-03-27 10:11:44"` |

> **注意**：根据 `sensor_type` 的不同，携带的参数不同。服务器会自动添加 `received_at` 字段（ISO 格式时间戳）并保存。

**响应示例**
```json
{
  "status": "success",
  "message": "数据接收成功",
  "filename": "data_20260327_101144_964873.json",
  "timestamp": "2026-03-27T10:11:44.964873"
}
