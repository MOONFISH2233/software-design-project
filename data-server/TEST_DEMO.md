## 🔍 详细功能演示

### 加密解密流程
```
1. 输入测试数据
   {"sensor": "temperature", "value": 25.5}
   
2. 点击"测试加密"
   → 获得加密字符串: "gAAAAABlM..."
   
3. 点击"测试解密"（自动启用）
   → 还原原始数据: {"sensor": "temperature", ...}
   
4. 对比验证数据一致性 ✓
```

### 认证流程
```
1. 输入 admin / admin123
2. 点击"测试登录"
   → 获得JWT Token
   
3. 点击"生成API Key"（自动启用）
   → 获得API Key: "key_admin_xxx"
   
4. 使用Token测试安全接口 ✓
5. 使用API Key测试认证接口 ✓
```
